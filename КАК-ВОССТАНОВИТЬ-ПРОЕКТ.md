# 🔄 Как поднять «Кинозал на двоих» заново (после переустановки Windows)

> Этот файл лежит на GitHub, поэтому переустановку он переживёт. Открой его на
> github.com/A1e0/kino, если что-то забудешь.

## ✅ Сначала главное: сайт НЕ сломается

Сайт **https://kino-kaez.onrender.com** работает на сервере Render (в интернете),
а фильмы лежат в облаке Cloudflare R2. **Ни то, ни другое не зависит от твоего
ноутбука.** Можешь спокойно переустанавливать Windows — сайт всё это время будет
онлайн, друг сможет смотреть. Восстанавливать проект на компьютере нужно только
если захочешь **что-то менять/добавлять**.

---

## Что нужно поставить заново (по порядку)

1. **Node.js** — https://nodejs.org → большая кнопка **LTS** → установить.
   (Это даёт `node` и `npm` — без них проект не запустится.)
2. **VS Code** — https://code.visualstudio.com → Download for Windows.
3. **Git** *(не обязательно, но удобно)* — https://git-scm.com/download/win

> Полный список всех твоих программ/языков/расширений — в файле
> `МОИ-ПРОГРАММЫ-перед-переустановкой.txt` (не забудь сохранить его на флешку!).

---

## Как скачать проект обратно

### Способ А (простой, без Git) — скачать ZIP
1. Открой https://github.com/A1e0/kino
2. Зелёная кнопка **Code** → **Download ZIP**.
3. Распакуй ZIP в удобную папку (например, на рабочий стол).

### Способ Б (через Git) — одной командой
```
git clone https://github.com/A1e0/kino.git
```

---

## Как запустить (всего 3 команды)

1. Открой папку проекта в VS Code: **File → Open Folder** → выбери папку `kino`.
2. Открой терминал: **Terminal → New Terminal** (или Ctrl + `).
3. Введи по очереди:
   ```
   npm install
   ```
   (скачает все нужные компоненты — express и ws — автоматически из `package.json`)
   ```
   npm start
   ```
4. Открой в браузере http://localhost:3000 — проект работает локально.

Готово! Всё, что нужно для работы, ставится одной командой `npm install`.

## Если захочешь снова заливать фильмы (rclone)

Инструмент `rclone` и его настройка (папка `_tools/`) на компьютере НЕ сохранятся —
но это не страшно, фильмы уже в облаке. Когда понадобится добавить новый фильм:
просто скажи Claude «добавь фильм» — он заново поставит rclone, попросит у тебя
новый ключ R2 (Cloudflare → R2 → Manage R2 API Tokens → Create) и всё сделает.
